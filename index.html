Swing Trading Strategy Checker

About this strategy

This tool identifies institutional-style buy setups by combining momentum reset and trend resumption.

Logic:

  • Momentum reset: RoC(10d) < 0
  • Trend resumption: Latest close > AVWAP (anchored at recent highest close)
  • PASS: both conditions true; otherwise FAIL
  • FAIL: if either condition is false
